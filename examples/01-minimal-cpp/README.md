## Hello World
This is the simplest example of using the libwk2gtkpdf to programatically generate a PDF from internally generated HTML.  

To run this example, copy this directory to your home folder:

- cp -r /usr/share/doc/libwk2gtkpdf-dev/examples/01-minimal-cpp ~/

Then run make and 

- ./hellopdf.

`./hellopdf` will generate 2 files 

- hellopdf.html 
 
And;

- hello.pdf

The cpp file itself is heavily documented.
